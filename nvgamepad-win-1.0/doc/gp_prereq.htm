NVIDIA GameWorks NvGamepad Prerequisites

Windows Development:

  • Windows Vista, 7, or 8 PC
  • Visual Studio 2010 or later, non-Express editions
  • Microsoft DirectX End-User Runtime Web Installer.
  • XINPUT-compatible gamepad

Android Development:

  • Windows Vista/7/8 host PC
  • Tegra 4-based or better device
  • External Android-supported game controller, NVIDIA SHIELD, or other Android device with a built-in or supplied game controller
  • Build environment:
    • Visual Studio 2010 or later, non-Express editions
    • Full Tegra Android Development Pack for Windows tools install
      (Note that NVIDIA Nsight Tegra, Visual Studio Edition should be included in full install if Visual Studio detected)
